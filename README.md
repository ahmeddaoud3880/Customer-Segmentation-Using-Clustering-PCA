# Customer Segmentation Using Clustering & PCA

## 📌 Project Overview
Customer segmentation is a key strategy used by businesses to better understand their customers and personalize services. This project applies unsupervised machine learning techniques to group customers into meaningful segments based on behavioral and demographic features.

Dimensionality reduction using Principal Component Analysis (PCA) was also applied to improve visualization and model efficiency.

---

## 🎯 Business Problem
Organizations often serve customers with different behaviors and needs. Without segmentation, companies may:

- Waste marketing budget
- Deliver irrelevant offers
- Lose potential revenue opportunities

This project helps businesses:

- Identify customer groups
- Design targeted marketing campaigns
- Improve customer experience
- Increase revenue through personalization

---

## 📊 Dataset
The dataset contains customer attributes such as demographics, spending behavior, and usage patterns.

## 📈 Results & Insights
The clustering algorithm successfully identified distinct customer groups with different characteristics and behaviors. PCA helped visualize clusters in lower dimensions, making patterns easier to interpret.

These insights can help companies design targeted marketing strategies and optimize customer engagement.
